# C-Adaptive-Upscaler-Plus
 - Upscales &amp; Enhances Videos, Gifs, and Images to the size you want
 - Increases quality

you need one of these
ARM64: https://aka.ms/vc14/vc_redist.arm64.exe
x86: https://aka.ms/vc14/vc_redist.x86.exe
x64: https://aka.ms/vc14/vc_redist.x64.exe

Settings can be changed in the "commands" json in the "ASSETS" folder
the "commands list" explains how to use the commands


Put your media in the "MEDIA" folder (you can do multiple for a queue, and it works for videos, gifs, and pictures)

Run the "AI Upscaler" file

A console will show the progress

When finished, your export will be in the "OUTPUT" folder
